Europe in Six Places

A small GIS story about European integration through six symbolic places. The story uses one point layer: european_integration_places.

This page is intentionally simple: each button calls the public GIS API directly.

Selected command: none

Europe as geography

A project made of places

European integration is often told through treaties, institutions and dates. This story reads it as a map: Rome, Schengen, Maastricht, Brussels, Strasbourg and Luxembourg.

Select a button to show the corresponding point or group of points on the GIS map.

Layer: european_integration_places Geometry: points

Layer fields

The GeoJSON layer contains these fields: id, name, country, year, phase, type, short_label, title, description.

API used by this story: GIS.showLayer(layerName, filter), GIS.showOnlyLayer(layerName, options), GIS.saveState(name), GIS.restoreState(name).
